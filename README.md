## What this is

A personal portfolio website built like a **cinematic showcase**, not a CV. Opens with a *Meet Danish* dossier intro and a cyber-warfare hero scene, then dissolves into 9 project scenes — each with its own bespoke entrance animation:

> MIRAX · HEIMDALL + AEGIS · GUARD · AURA · SPIDY · CORIZO · CyberMentor · Neural Lock · L&T × Sony

Recruiters skim portfolios in 30 seconds. A cinematic, interactive site makes them **stop and watch** — turning a "meh" click into a memorable impression and a callback.

**🌐 Live:** [danish-spidy.github.io/danish-portfolio](https://danish-spidy.github.io/danish-portfolio/)

## Run locally

```bash
git clone https://github.com/Danish-spidy/danish-portfolio
cd danish-portfolio
python -m http.server 8181
# open http://localhost:8181 in Chrome
```

## Stack

- **HTML + CSS + JavaScript** — no framework, max speed
- **GSAP** for animation orchestration
- **Custom motion logic** for each project scene

Hosted free on GitHub Pages — [danish-spidy.github.io/danish-portfolio](https://danish-spidy.github.io/danish-portfolio/).
